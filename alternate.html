What is Thrift?

Thrift is a decentralized exchange protocol that works without an order book. Instead of an order book, it utilizes a model called automated market-making (AMM), where liquidity providers add funds to liquidity pools. Liquidity providers (LP) add an equal value of two tokens in a pool to create a market. Wallet Connect Feature

Wallet connect feature avaliable which allows you to connect wallet easily and you can transfer your crypto into your wallet. WalletConnect is an open source protocol for connecting decentralised applications to mobile wallets with QR code scanning or deep linking. A user can interact securely with any Dapp from their mobile phone, making WalletConnect wallets a safer choice compared to desktop or browser extension wallets.
